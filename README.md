# Hello! I'm Daniel Gualán

I am student software engineer focused on operating systems, infrastructure behavior, and disciplined software design.

My work revolves around understanding how systems behave under real conditions — how processes interact, how networks expose boundaries, and how architectural decisions affect resilience and control.

I am particularly interested in infrastructure discipline, reproducible environments, controlled deployments, and reducing unnecessary complexity across systems.

Currently deepening knowledge in:

- Operating Systems internals  
- Computer Networks (TCP/IP, segmentation logic)  
- System Architecture & behavior modeling  
- Structured build lifecycles (Maven)  
- Container isolation models  
- Observability and controlled CI/CD pipelines  

I am continuously refining fundamentals before narrowing specialization.
